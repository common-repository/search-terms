== Description ==

Improve the chances of converting searches into sales with the Search Terms plugin for WordPress.
This powerful tool allows you to easily add synonyms or redirects to your search terms, providing users with a more comprehensive and accurate search experience.
Additionally, you can track the popularity and number of results for each search query, giving you valuable insights into what your users are looking for and how to optimize your content for better results.
Don't miss out on this essential tool for maximizing the potential of your WordPress search feature.

== Installation ==

1. Upload the entire contents of the zip file to your plugin directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin from Wp Admin => Search Terms
